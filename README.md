## 💻 Projeto

Esse é um projeto que visa ajudar listando saus compras. Contém as seguintes funcionalidades: Adicionar itens á lista de compras , remover itens da lista de compras, marcar os itens que ja foram pegos com o texto taxado.

## Como usar?

Baixe este repositório e com seu teminal, entre no diretório

1. Abra este projeto com seu editor favorito;
2. Depois, instale a extensão Live Server; 
3. Em seguida, abra o index.html com o Live Server para ver o projeto.
